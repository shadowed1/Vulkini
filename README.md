# Goal: Achieve the best performance that Crostini will allow.

1. Back up your VM or don't
2. Go to Chrome://flags and enable GPU enabled
3. Start Crostini/Penguin
4. Copy paste in Terminal:
```
sudo curl -fsSL "https://raw.githubusercontent.com/shadowed1/Vulkini/main/bin/install_vulkan.sh" -o "/bin/install_vulkan"
sudo chmod +x "/bin/install_vulkan"
install_vulkan
```
